# MiniBlog Escolar — Demonstração XSS

Protótipo web desenvolvido para a disciplina de Segurança da Informação.

## Objetivo

Demonstrar, de forma didática e controlada, a diferença entre uma página vulnerável a XSS e uma página corrigida.

## Aviso ético

Este projeto possui finalidade exclusivamente educacional.  
Nenhum site real, sistema de terceiros, dado pessoal ou serviço externo foi atacado, explorado ou comprometido.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como usar

1. Acesse a página publicada pelo GitHub Pages.
2. Insira um comentário comum ou use o exemplo didático de XSS.
3. Compare o comportamento entre a versão vulnerável e a versão corrigida.

## Observação

A demonstração é simulada e segura. O objetivo é representar o conceito de XSS sem realizar ataques reais.
